## 技術認知總結

### 1. SIMPLE vs PJFNK 的本質差異

| 面向 | PJFNK | SIMPLE |
|------|-------|--------|
| 矩陣 | 全耦合 4N×4N | 分離式 4×(N×N) |
| 每步時間 | ~1000 秒 | 預期 ~數秒 |
| 適合場景 | 瞬態精確 | 穩態啟動 |
| 對 mu_t | Jacobian 不包含 → Picard 退化 | 天然分離，穩定 |

### 2. MSFR 啟動困難的根本原因

- Re = 4.8×10⁴（湍流流動）
- Stokes 初始場：速度≈0 → mu_t≈0 → 等效層流 Re=48000
- SIMPLE 在高 Re 無阻尼下無法穩定
- 解法：給 mu_t 一個合理的非零初始值提供數值阻尼

### 3. 壓力方程式的正確寫法（SIMPLE）

```
# 錯誤（用於 transient/全耦合）：
INSFVMassAdvection

# 正確（SIMPLE 穩態）：
FVAnisotropicDiffusion  # ∇⋅(Ainv ∇p')
FVDivergence            # ∇⋅(HbyA)
```

### 4. 物理時間尺度（重要教訓）

- MSFR 流速 1.18 m/s，爐心直徑 2.3m
- 滯留時間 ≈ 1~2 秒
- 需要推進到 t~1s 才能建立真正的對流場
- Transient PJFNK 在此問題不可行（133天）
