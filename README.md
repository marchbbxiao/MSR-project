# MSR-project

Replication of Tano et al. 2025 coupled MSR simulation framework.

## 結構
- `msre-project/` — MSRE OpenMC benchmark（Stage 1，k-eff = 1.00637 ± 0.00113）
- `MSFR/` — MSFR standalone OpenMC + Cardinal coupling（Stage 2+，待遷移）
- `工作日誌/` — 所有開發日誌與思維記錄
  - `msre/` — MSRE 階段日誌
  - `msfr/` — MSFR 階段日誌（待遷移）
